      Get auto-refresh

      get(string $sid) 
      API

      Retrieve current page auto-refresh time. Must be logged in to use.

      api

      Parameters

      $sid

      string

      Session ID that is provided when logged in. This is also set as a cookie. Only required if the cookie sid header is not sent.

      Remove auto-refresh

      remove(string $sid) 
      API

      Removes the auto-refresh time. This is the same as setting /refresh/set to 0 seconds.

      api

      Parameters

      $sid

      string

      Session ID that is provided when logged in. This is also set as a cookie. Only required if the cookie sid header is not sent.

      The time in seconds for the auto refresh. Defaults to 10 seconds.
